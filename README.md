# ScreenRec

## English
AutoHotkey screen recording script with a small desktop interface, a timer, and integration with ffmpeg.

### Features
- Start recording with a short countdown
- Stop recording from the UI
- Save recordings to the desktop
- Open the recordings folder quickly

### Requirements
- AutoHotkey
- ffmpeg.exe available next to the script

### Usage
1. Place ffmpeg.exe in the same folder as the script.
2. Run ScreenRecorder.ahk.
3. Start recording from the interface.
4. Stop the recording and check the generated file on the desktop.

## Português
Script de gravação de tela em AutoHotkey com interface pequena, cronômetro e integração com ffmpeg.

### Funcionalidades
- Iniciar a gravação com uma contagem curta
- Parar a gravação pela interface
- Salvar as gravações na área de trabalho
- Abrir rapidamente a pasta das gravações

### Requisitos
- AutoHotkey
- ffmpeg.exe disponível ao lado do script

### Uso
1. Coloque ffmpeg.exe na mesma pasta do script.
2. Execute ScreenRecorder.ahk.
3. Inicie a gravação pela interface.
4. Pare a gravação e confira o arquivo gerado na área de trabalho.
